LaTeX for cohost!!

do you like chosting? do you like 𝐿𝐴𝑇𝐸𝑋? you can now combine these two passions using this tool i made. you can convert all sorts of funky formulas to cohost-compatible HTML:

𝜑=1+11+11+11+⋯

{⟨𝑥,𝑦⟩:𝑥,𝑦∈ℕ,𝑊𝐾𝑥⊆𝑊𝐾𝑦}

𝐼4×4=⎡⎣⎢⎢⎢⎢1000010000100001⎤⎦⎥⎥⎥⎥

How does it work? this tool uses MathJax v. 2.7 to turn LaTeX into HTML.

the whole thing is only ~80 lines of javascript.

surprisingly, most browsers seem to have reasonably good fonts for "STIXGeneral" and stuff, so you don't actually need MathJax's fonts for the most part.

How many bytes does it use? as you may know, cohost has a 200kb maximum post size. mathjax can generate pretty long HTML. for reference, this post is 19473 bytes long so you should be fine, at least for anything short-ish.
Do the formulas look good in different browsers? so far I've tested Firefox, Chromium, Android Firefox, and Android Chrome, and they all look fine. The mobile browsers display them a bit wonky but they're still readable.

integrals, square roots, and \left/\right all look bad on windows/mobile but look good on linux. i'm not sure why.

Is it accessible? unfortunately most of mathjax's accessibility features would be removed by cohost. most formulas aren't going to be read very well by a screenreader, but it's probably better than nothing.
