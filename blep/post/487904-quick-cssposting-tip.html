oh no! my css crime is clipping into the read more button!

that’s because some element has a css z-index higher than those elements. sometimes you can just simply set your z-indices lower (e.g. the top bar is at z-index 10) but the “read more” is at a z-index of 0… what do? don’t worry! there is a better solution:

isolation: isolate

this is an obscure css property that creates a new stacking context inside an element! add this to whatever <div> contains your css crime and it will magically confine all elements to within that context.
